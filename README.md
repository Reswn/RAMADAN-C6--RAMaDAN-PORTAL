# 🌙 Ramadhan Space 2026 - Portal Ibadah Digital

[![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen?style=for-the-badge)](https://ramadan-c7-ra-ma-dan-portal-6jmb.vercel.app/)
[![License](https://img.shields.io/badge/License-MIT-blue?style=for-the-badge)](LICENSE)
[![Status](https://img.shields.io/badge/Status-Completed-success?style=for-the-badge)]()

**Ramadhan Space** adalah portal ibadah digital modern yang dirancang untuk menemani perjalanan spiritual umat Muslim selama bulan suci Ramadhan 1447 H / 2026 M. Platform ini menyediakan berbagai fitur terlengkap dalam satu halaman, mulai dari jadwal imsakiyah real-time, kumpulan doa, penghitung zikir, kalkulator zakat, hingga artikel islami inspiratif.

Dibangun dengan pendekatan *mobile-first* dan desain antarmuka yang estetis, Ramadhan Space bertujuan membuat ibadah menjadi lebih terorganisir, konsisten, dan khusyuk.

---

## ✨ Fitur Utama

### Dashboard & Navigasi
- **Desain Responsif:** Tampilan optimal di Desktop, Tablet, dan Mobile.
- **Dark Mode:** Mode gelap yang nyaman untuk ibadah malam (Tarawih & Sahur).
- **Navigasi Intuitif:** Menu sidebar dan navbar yang mudah diakses.

###  Jadwal Imsakiyah
- Jadwal sholat dan imsak otomatis berdasarkan lokasi.
- Countdown menuju waktu sholat berikutnya.
- Indikator hari keberapa puasa saat ini.

### Fitur Ibadah
- **Kumpulan Doa:** Doa sahur, buka puasa, dan doa harian lengkap dengan Arab, Latin, dan Arti.
- **Digital Tasbih (Zikir):** Penghitung zikir interaktif dengan target harian.
- **Kalkulator Zakat:** Menghitung zakat fitrah dan zakat maal secara otomatis sesuai nisab.
- **Ibadah Tracker mencatat ibadah harian anda untuk memantau progress

### Konten & Edukasi
- **Artikel Islami:** Artikel tentang keutamaan Ramadhan, tips kesehatan saat puasa, dan kisah teladan sahabat.
- **Tips Kesehatan:** Panduan menjaga stamina tubuh selama berpuasa.

### 2. Panduan Fitur

####  **Beranda (Dashboard)**
Halaman utama dengan ringkasan ibadah dan navigasi cepat.

```
┌─────────────────────────────────────────┐
│  🌙 Ramadhan Space                      │
│  ─────────────────────────────────────  │
│  [Beranda] [Jadwal] [Doa] [Zikir] ...  │
│                                         │
│  ┌──────────┐ ┌──────────┐             │
│  │  Jadwal  │ │  Doa     │             │
│  │  Sholat  │ │  Harian  │             │
│  └──────────┘ └──────────┘             │
└─────────────────────────────────────────┘
```

**Cara Menggunakan:**
1. Klik menu di navbar untuk berpindah halaman
2. Scroll ke bawah untuk melihat fitur unggulan

---

#### **Jadwal Imsakiyah**
Jadwal sholat otomatis berdasarkan lokasi dengan countdown real-time.

**Langkah Penggunaan:**
```
1. Buka halaman "Jadwal Imsak"
2. Pilih kota Anda dari dropdown 
3. Lihat jadwal sholat hari ini
4. Countdown otomatis menuju sholat berikutnya
5. Export jadwal ke CSV jika perlu 
```

**Fitur:**
- Auto-detect lokasi
- Countdown real-time
- Highlight hari ini
- Export ke Excel/CSV

---

#### **Kumpulan Doa**
Doa sahur, buka puasa, dan doa harian lengkap dengan Arab, Latin, dan Arti.

**Cara Menggunakan:**
```
1. Buka halaman "Doa"
2. Pilih kategori doa (Sahur / Buka )
3. Klik kartu doa untuk melihat detail
4. Baca Arab, Latin, dan Arti
5. Salin teks jika diperlukan 
```

**Kategori Tersedia:**
| Kategori | Jumlah Doa |
|----------|-----------|
| Doa Sahur | 2+ |
| Doa Buka Puasa | 1+ |
| Doa Ramadhan | 2+ |

---

####  **Digital Tasbih (Zikir)**
Penghitung zikir interaktif dengan target harian dan statistik.

**Cara Menggunakan:**
```
1. Buka halaman "Zikir"
2. Pilih jenis zikir dari daftar:
   • Subhanallah (33x)
   • Alhamdulillah (33x)
   • Allahu Akbar (33x)
   • La ilaha illallah (100x)
   • Astaghfirullah (100x)
   • Dan lainnya...
3. Klik tombol "Tambah" atau tekan SPASI
4. Progress bar otomatis terisi
5. Notifikasi saat target tercapai 
6. Reset untuk zikir berikutnya
```

**Tips:**
- Tekan **SPASI** untuk menghitung cepat
- Data tersimpan otomatis di browser
- Lihat statistik total zikir dan streak

---

#### **Kalkulator Zakat**
Hitung zakat fitrah dan zakat maal secara otomatis sesuai nisab.

**Cara Menggunakan:**
```
1. Buka halaman "Kalkulator Zakat"
2. Pilih jenis zakat:
   □ Zakat Penghasila
   □ Zakat Perusahaan (Jasa)
   □ Zakat Perusahaan  (Industri/Dagang)
   □ Zakat Perdagangan 
   □ Zakat Emas
3. Masukkan data yang diperlukan
4. Klik "Hitung Zakat"
5. Lihat hasil perhitungan
6. Export atau cetak hasil 📄
```

**Rumus Perhitungan:**
```
Zakat Fitrah = Jumlah Jiwa × Harga Beras × 2.5kg
Zakat Maal   = Total Harta × 2.5% (jika已达 nisab)
```

---

#### **Ibadah Tracker**
Catat dan pantau progress ibadah harian Anda selama Ramadhan.

**Cara Menggunakan:**
```
1. Buka halaman "Ibadah Tracker"
2. Login/daftar (data tersimpan lokal)
3. Checklist ibadah harian:
   □ Shalat 5 waktu
   □ Baca Qur'an (target halaman)
   □ Puasa hari ini
   □ Dzikir pagi & petang
4. Lihat progress bar harian
5. Pantau streak ibadah berturut-turut 
6. Export laporan bulanan
```

**Fitur Tracking:**
- Progress harian (%)
- Streak contador
- Kalender puasa 1-30
- Statistik mingguan

---

#### 📰 **Artikel Islami**
Baca artikel tentang keutamaan Ramadhan, tips kesehatan, dan kisah teladan.

**Cara Menggunakan:**
```
1. Buka halaman "Artikel"
2. Browse kategori:
   • Keutamaan Ramadhan
   • Tips Kesehatan Puasa
   • Kisah Sahabat
   • Fiqih Puasa
3. Klik artikel untuk baca detail
4. Komentar dan Share ke media sosial (fitur to be continued karena harus akses database)
```

### 3. Fitur Global

#### **Dark Mode**
Mode gelap untuk kenyamanan ibadah malam (Tarawih & Sahur).

**Cara Mengaktifkan:**
```
Desktop: Klik icon 🌙 di navbar kanan
Mobile:  Buka sidebar → Klik "Mode Gelap"
Shortcut:  Data tersimpan otomatis
```

**Keuntungan:**
- Nyaman untuk mata di malam hari
- Hemat baterai (OLED screen)
- Preferensi tersimpan otomatis

#### **Navigasi Mobile vs Desktop**

| Fitur | Desktop | Mobile |
|-------|---------|--------|
| **Menu** | Navbar horizontal | Hamburger menu ☰ |
| **Sidebar** | Selalu terlihat | Slide dari kanan |
| **Dark Mode** | Icon di navbar | Tombol di sidebar |
| **Layout** | 3 kolom | 1 kolom (stack) |

---

#### **Penyimpanan Data**

Semua data disimpan di **LocalStorage** browser Anda:

```
┌─────────────────────────────────────┐
│  Data Tersimpan:                    │
│  ✓ Preferensi Dark Mode            │
│  ✓ Progress Ibadah Tracker         │
│  ✓ Counter Zikir                   │
│  ✓ Target Qur'an                   │
│  ✓ Kota Pilihan (Jadwal)           │
└─────────────────────────────────────┘
```

**Catatan:** Data tidak hilang saat refresh, tetapi akan hilang jika clear cache browser.

### 4. Tips Penggunaan

| Tips | Deskripsi |
|------|-----------|
| **Sahur** | Gunakan dark mode, cek jadwal imsak |
| **Buka** | Set notifikasi, baca doa buka puasa |
| **Tarawih** | Dark mode, hitung zikir setelah sholat |
| **Tracking** | Update Ibadah Tracker setiap hari |
| **Zakat** | Hitung zakat fitrah sebelum Idul Fitri |
| **Belajar** | Baca artikel untuk tambah ilmu |

---

### 5. Troubleshooting

| Masalah | Solusi |
|---------|--------|
| Jadwal tidak muncul | Izinkan akses lokasi, refresh halaman |
| Dark mode tidak tersimpan | Clear cache, coba browser lain |
| Data Ibadah hilang | Jangan clear localStorage browser |
| Tampilan rusak di mobile | Refresh, update browser ke versi terbaru |
| Audio notifikasi tidak bunyi | Cek izin notifikasi browser |

---

### 6. Shortcut Keyboard

| Tombol | Fungsi |
|--------|--------|
| `SPASI` | Tambah counter zikir |
| `R` | Reset counter zikir |
| `ESC` | Tutup sidebar mobile |
| `D` | Toggle dark mode (beberapa halaman) |

##  Teknologi yang Digunakan

Proyek ini dibangun menggunakan teknologi web standar yang ringan dan cepat:

- **HTML5:** Struktur semantik untuk SEO yang baik.
- **CSS3:** Styling custom dengan variabel CSS, Flexbox, Grid, dan animasi halus.
- **JavaScript (Vanilla):** Logika interaktif untuk countdown, dark mode, kalkulator, dan penyimpanan lokal (LocalStorage).
- **Font Awesome & Bootstrap Icons:** Ikon vektor untuk elemen visual.
- **Google Fonts:** Tipografi modern (Poppins & Amiri).
- **Vercel:** Platform hosting untuk deployment yang cepat dan stabil.
